\doxysubsection{Detailed Description}
Used to store the graphs for the multiple cycles of a single element. These graphs are derived from a coresponding integral histogram. EX\+: If we are doing 20 cycles of 20 runs for the decay chain 144Cs-\/$>$144Ba-\/$>$144La this class would hold all the histograms for the 20 cycles of La, each cycle held by a \mbox{\hyperlink{classRunHistoHolder}{Run\+Histo\+Holder}} class.
