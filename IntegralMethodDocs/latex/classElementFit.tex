\doxysubsection{Detailed Description}
This class can be considered the cornerstore for the entire program. This class is the class that generated the data then fits it. After the generation and fitting, the fitted values are passed onto the \mbox{\hyperlink{classRun}{Run}} class if multiple Runs/\+Cycles are chosen.
