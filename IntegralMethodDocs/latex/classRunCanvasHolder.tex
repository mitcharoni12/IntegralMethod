\doxysubsection{Detailed Description}
Used to store the canvases which will display each individual histogram if the user wishes to display each individual histogram. This class stores the canvases of each run in one cycle for a single element. EX\+: if we do 20 cycles of 20 runs of the decay chain 144Cs-\/$>$144Ba-\/$>$144 La there are 20 histograms per cycle. This class would store the canvases for one cycle of La.
