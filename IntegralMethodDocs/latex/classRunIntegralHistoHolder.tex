\doxysubsection{Detailed Description}
Used to store all the histograms for the multiple integral fits in a single cycle for a single element. These histograms are the ones filled with simulated data then fitted. EX\+: If we are doing 20 cycles and 20 runs per cycles of the decay chain 144Cs-\/$>$144Ba-\/$>$144La. This would hold the 20 histograms of a single cycle for the element La.
