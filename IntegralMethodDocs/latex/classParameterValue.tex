\doxysubsection{Detailed Description}
Holds N0, Half-\/life, and Decay constant taken from the user for a single element in the decay chain. These values are the inital parameters for the fit functions are the values used to generate the simulated data. Can also take a range for these values if a user wants to set a specific range on the fit values. Currently in the program the fit parameter boundries are just multiples of these values.
