\doxysubsection{Detailed Description}
Used to store the histograms for the multiple Bateman cycles of a single element. These histograms contain the simulated data and the ones that are fit. EX\+: If we are doing 20 cycles of 20 runs for the decay chain 144Cs-\/$>$144Ba-\/$>$144La this class would hold all the histograms for the 20 cycles of La, each cycle held by a \mbox{\hyperlink{classRunHistoHolder}{Run\+Histo\+Holder}} class.
