\doxysubsection{Detailed Description}
Used to store all the graphs for the multiple runs in a single cycle for a single element. These graphs are derived from a coresponding integral histogram. EX\+: If we are doing 20 cycles and 20 runs per cycles of the decay chain 144Cs-\/$>$144Ba-\/$>$144La. This would hold the 20 graphs of a single cycle for the element La.
