Detailed Description

Used to store the canvases which will display each individual histogram if the user wishes to display each individual histogram. This class stores the the canvases for the cycles and runs of a single element. EX: If we are doing 20 cycles of 20 runs for the decay chain 144Cs->144Ba->144La for each element we will have 400 histograms to store. This class would store the histograms for the 20 cycle of 20 runs for the element La.
