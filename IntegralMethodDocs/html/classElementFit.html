Detailed Description

This class can be considered the cornerstore for the entire program. This class is the class that generated the data then fits it. After the generation and fitting, the fitted values are passed onto the Run class if multiple Runs/Cycles are chosen.

The constructor for simualting data.

Will take the fit fuctions for the program and the fit options for both the Run class and Cycle class. Dynamically allocates the fit parameter storages here. Creates data required for creating the histograms between runs and cycles. Runs the call for creating all the histograms.
