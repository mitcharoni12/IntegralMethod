void Run::RunSingleBatemanRunsGenOnce ( Int_t  cycleIndex,
Int_t  runIndex 
)

Preforms the multiple runs of the program for the single bateman data not changing anything setting between runs and only generating a single set of histograms to fit with for each cycle.

Only used with the cycle program execution type. It would not make sense to do this with the run program execution type becuase then it would be the same as just generating and fitting a single histogram.
